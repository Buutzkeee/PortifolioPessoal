Como foi implementado

O Gerenciador de Tarefas foi desenvolvido utilizando HTML, CSS e JavaScript puro para garantir performance e simplicidade. A interface foi construída com CSS moderno para um design responsivo e elegante.

A funcionalidade CRUD (Criar, Ler, Atualizar, Deletar) foi implementada utilizando JavaScript para manipulação do DOM e gerenciamento de estado local. Cada operação é representada por funções específicas que atualizam a interface do usuário.

Tecnologias utilizadas

  • •
    HTML: Estruturação semântica e acessível do conteúdo.
  • •
    CSS: Estilização moderna com flexbox, grid e animações.
  • •
    JavaScript: Manipulação do DOM e gerenciamento de estado.
  • •
    LocalStorage: Persistência de dados no navegador.

O que aprendi

Este projeto me permitiu aprofundar conhecimentos em JavaScript puro, especialmente em relação ao gerenciamento de estado e manipulação do DOM. Aprendi como implementar operações CRUD de maneira eficiente e responsiva sem depender de frameworks.

Também pude praticar a criação de interfaces interativas com CSS moderno, utilizando grid e flexbox para layouts responsivos, além de implementar feedback visual para melhorar a experiência do usuário.

Um desafio interessante foi criar formulários dinâmicos para adição e edição de tarefas, garantindo a validação adequada e feedback ao usuário em tempo real.
